### Tone

### Description for Media (25 words)

### Description for Media (50 words)

> The AnubisX Framework introduces a paradigm shift in digital identity attribution — from technical to human-centric. By measuring five independent behavioral modalities through rigorous mathematical methods, it establishes Identity Intelligence as a formal forensic discipline with quantified uncertainty and Daubert-compliant standards.

## Key Messages

1. **Paradigm shift**: From technical artifacts to human cognitive signatures
2. **Scientific discipline**: Establishing Identity Intelligence alongside DNA, fingerprints, ballistics
3. **Multi-modal**: Five independent behavioral signals fused for high-confidence attribution
4. **Quantified**: Every conclusion includes measured uncertainty
5. **Ethical**: Privacy, proportionality, and dual-use safeguards built into the architecture
