## Getting Started

The AnubisX Framework is a theoretical specification ready for implementation. This guide helps developers understand the framework for implementation.

## Implementation Priority

1. **Core algorithms** — Feature extraction (ALG-001—020)
2. **Profile system** — Profile construction and storage
3. **Similarity computation** — Profile comparison (ALG-024—028)
4. **Calibration and fusion** — LR computation and fusion (ALG-021—023, 029—032)
5. **Decision and reporting** — Attribution decisions (ALG-033—036, 041)
