## Five Behavioral Modalities

### Forensic Stylometry
Measures involuntary linguistic patterns — function-word frequencies, syntactic structures, vocabulary richness. Resistant to topic variation and conscious manipulation.

### Chrono-Profiling
Analyzes temporal behavior — circadian rhythms, inter-event timing, response time patterns. Highly individual-specific and difficult to modify.

### Terminal Execution Profiling
Captures human-machine interaction patterns — command sequences, navigation paths, input timing. Reflects deeply ingrained procedural habits.

### Relational Network Analysis
Maps social graph structure, communication patterns, and relationship dynamics. Network topology provides rich behavioral signals.

### Environmental Media Forensics
Analyzes digital environment organization — file structures, naming conventions, metadata patterns. Reflects organizational habits.

## Fusion Architecture

Evidence from all five modalities is integrated through a structured pipeline:

1. **Feature Extraction** — Modality-specific behavioral features
2. **Profile Construction** — Individual behavioral profiles
3. **Profile Comparison** — Similarity scoring
4. **Evidence Integration** — LR calibration and multi-modal fusion
5. **Decision** — Attribution under quantified uncertainty

## Standards

- Daubert-compliant evidence framework
- Likelihood Ratio quantification
- Empirical cross-entropy validation
- Standardized benchmarks and baselines
