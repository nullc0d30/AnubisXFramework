## Repository Preservation Strategy

### Long-term Access Layers

1. **Active Layer** — GitHub for current development
2. **Archival Layer** — Zenodo for permanent preservation
3. **Academic Layer** — Journal publication for peer validation
4. **Discovery Layer** — Google Scholar/OpenAlex for indexing
5. **Social Layer** — ResearchGate/Figshare for community engagement

### Preservation Guarantees

- **GitHub:** Active maintenance while project is alive
- **Zenodo:** Permanent archival with CERN backing
- **Crossref:** Persistent identifier resolution
- **Google Scholar:** Long-term discoverability
- **OpenAlex:** Open knowledge graph preservation
